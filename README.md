## Overview

Mp3tag Professional Suite is a complete audio metadata platform with scripting, custom web sources, and export templates for professional library management.

Custom web source scripts connect to any online database that provides music metadata, extending lookup capability beyond the built-in sources.

Export templates generate HTML, CSV, and text reports from tag data for library documentation and catalog publishing.

Regex support in actions enables pattern-based tag manipulation for complex formatting requirements.

Mp3tag Professional Suite is the audio metadata platform for professionals who need comprehensive tag editing with the scripting, lookup, and export tools that production-scale library management demands.

## Performance & Integration

Mp3tag is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
